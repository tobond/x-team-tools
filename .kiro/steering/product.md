# Product Overview

## x-team-tools

A collection of common useful tooling for development teams, including scripts, AI prompts, templates, and development environment orchestration. The project focuses on providing standardized tools and processes that enhance developer productivity and maintain consistency across teams.

## Key Components

- **AI Prompts**: Standardized prompts for code review and development assistance
- **Requirements Templates**: Structured approach to feature specification and requirements gathering
- **Development Environment**: Tilt-based local Kubernetes development environment supporting multiple technology stacks (Python, Java, Go, Node.js, CrewAI services)
- **Process Documentation**: Guidelines for requirements creation, code review, and development workflows

## Target Users

- Software developers working on multi-service applications
- Team leads managing development processes
- DevOps engineers setting up local development environments
- Product managers creating feature specifications

## Business Value

Reduces development friction by providing standardized tools, templates, and environments that enable faster iteration cycles and consistent quality across teams.
